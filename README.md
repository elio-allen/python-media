# Python media! Elio's proprietary FOSS media format.  

Python media was made as a fun project on Elio's spare time. Python media supports 3 essoterric forms of that Elio created formats for:
* [Images](https://github.com/elio-allen/python-media/blob/main/image_viewer.py)
* [Videos](https://github.com/elio-allen/python-media/blob/main/anim_viewer.py)
* [Audio](https://github.com/elio-allen/python-media/blob/main/anim_audio_viewer.py)  
  
Each file is the same as the last but with added content!  
I started with the images then made many images then added audio to make video + audio!  

I've included some examples to [this repo](https://github.com/elio-allen/python-media/tree/main/examples) so you can learn how it works by seeing it.

If you're running [elio2.txt](https://github.com/elio-allen/python-media/tree/main/examples/elio2.txt), you will need to use a minimum of v2 and for [anim.txt](https://github.com/elio-allen/python-media/blob/main/examples/anim.txt) a minimum of v3. If you run these on older versions, they will fail because they dont have video or audio implementation.  
But you can run a v1 media file on v3 software.  

For this project, there's two modes. shell and GUI. If you are running it in IDLE I reccomend using the GUI version or you will get a big mess. If you're running this headless, you can run it in your shell but you may not have audio.

Make sure to check [requirements.txt](https://github.com/elio-allen/python-media/blob/main/requirements.txt) to be sure you have all the required packages
